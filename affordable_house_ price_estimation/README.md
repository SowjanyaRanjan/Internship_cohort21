This application is a machine learning–based web solution developed to estimate house prices in Bengaluru. It allows users to enter details such as location, 
number of bedrooms (BHK), number of bathrooms, and total square footage to get an estimated property price.The goal of building this system was to combine data 
analysis, model building, and user interface design into a single working application that is simple to use and practically useful.

The prediction model was trained on a cleaned Bengaluru housing dataset. After preprocessing and training, the model was saved and integrated into a Streamlit-based 
web interface.When a user enters the required details and clicks the prediction button, the app processes the inputs and displays the estimated price in a clear and 
readable format.
